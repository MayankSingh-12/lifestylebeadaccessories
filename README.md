## Adding Images

To add images: upload `.jpg` or `.png` files to the `/images/` folder.

The website expects these filenames:

| File | Used In |
|------|---------|
| `image1.jpg` | Collection gallery — top left |
| `image2.jpg` | Collection gallery — top right |
| `image3.jpg` | Collection gallery — middle left |
| `image4.jpg` | Collection gallery — middle right |
| `image5.jpg` | Collection gallery — full width banner |
